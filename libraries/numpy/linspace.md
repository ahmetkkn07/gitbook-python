# linspace

Başlangıç ve bitiş değeri arasındaki değerleri, istenen adet kadar parçaya böler.

```python
import numpy as np


x = np.linspace(0, 100, 5)
print(x)
# [  0.   25.   50.   75.  100.]


y = np.linspace(0, 1, 6)
print(y)
# [0.  0.2  0.4  0.6  0.8  1. ]


z = np.linspace(0, 10, 8)
print(z)
# [ 0.  1.42857143  2.85714286  4.28571429  5.71428571
#   7.14285714  8.57142857  10.]
```
